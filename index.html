COMO DESCREVER UM SHADER
Prompt ideal para qualquer chat:

"Crie um fragment shader GLSL para WebGL1 audioreativo.
Uniforms disponíveis: uTime, uBass, uMid, uTreble (0-1),
uRes (vec2), uTex (sampler2D), uHasTex (int),
uInt, uSpd, uZoom, uDist, uMix.
Efeito: [DESCREVA AQUI — ex: espiral fractal que pulsa com o bass]
Retorne SOMENTE o codigo GLSL, sem markdown."
Depois use COLAR SHADER para importar a resposta diretamente.
IMPORTAR RESPOSTA DO CHAT
1. Copie o codigo GLSL gerado pelo chat
2. Clique em COLAR SHADER no painel de IA
3. Cole o codigo e clique em Importar
O app adapta automaticamente os uniforms.
Chaves salvas apenas no navegador (localStorage). Nunca enviadas a terceiros exceto a propria API escolhida.
